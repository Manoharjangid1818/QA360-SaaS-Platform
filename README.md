# ?? QA360 – Smart QA Monitoring Tool

A real-time QA automation and monitoring system built using Playwright and Node.js.

## ?? Features
- Automated end-to-end testing
- Multi-test execution (login, cart, logout, etc.)
- Real-time PASS/FAIL status
- Dashboard to trigger tests
- Logs generation system

## ?? Tested On
- SauceDemo (Demo application)

## ?? Tech Stack
- Playwright
- Node.js
- Express
- HTML/CSS

## ?? How to Run

npm install
npx playwright install
node server/app.js

Open:
http://localhost:5000

## ?? Test Scenarios
- Login Test
- Product Validation
- Add to Cart
- Logout

## ?? Purpose
To build a SaaS-style QA monitoring system instead of simple scripts.
